# warp-generator 
Một tập lệnh có thể tạo ra lưu lượng Warp+ không giới hạn và hoàn toàn miễn phí!
# Cách Dùng:

Tập lệnh này có thể hoạt động cả trên Windows lẫn Android (Termux), dưới đây là hướng dẫn:

```

pip install requirements.txt

python ndbiaw.py

```
